## Stack technologiczny

- Node.JS
- React
- Typescript
- MongoDB
- typeless - mój autorski framework do Reacta

## O Projekcie

Robie kompletny poker room, gdzie będzie można grać w pokera Texas Holden No Limit na wirtualne pieniądze.

## Zasady live codingu

- Cały kod jest pisany na bieżąco.
- Nic nie przygotowuje przed nagrywaniem i nie odtwarzam wcześniej zapisanego kodu z pamięci.
- Architektura i design aplikacji, też jest robiony spontaniczne.
